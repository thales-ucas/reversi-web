# reversi


苹果棋 黑白棋


## 支持库


threejs


## 算法


蒙特卡洛树


## 游戏结构


使用vue创建前端结构


构建reversi游戏类


游戏的数据全在engine(游戏引擎)里，main负责逻辑结构调用


board(棋盘)，square(方格)，piece(棋子)
